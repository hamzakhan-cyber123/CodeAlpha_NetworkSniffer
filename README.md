# 🔒 Basic Network Sniffer — CodeAlpha Internship Task 1

A Python-based network packet sniffer built using Scapy.
Captures live network traffic and displays source/destination IPs,
protocols (TCP/UDP/ICMP), ports, and payloads in real time.

## Features
- Live packet capture using Scapy
- Protocol detection: TCP, UDP, ICMP
- Source & Destination IP + Port display
- Payload extraction (UTF-8 decoded)
- Clean terminal output with timestamps

## Requirements
- Python 3
- Scapy library
- Kali Linux (root access required)

## How to Run
sudo python3 network_sniffer.py
